# Hostel Frontend Room Booking Project

## Project Overview

This project is a Hostel Frontend Room Booking System developed to manage hostel room bookings efficiently. It provides students and administrators with an easy-to-use platform for room allocation, booking management, and hostel information handling.

## Features

* Student Registration and Login
* Room Booking Management
* Hostel Room Availability Check
* Warden Information Management
* Student Profile Management
* Admin Dashboard
* Booking Status Tracking
* Responsive Frontend Design

## Technologies Used

* ASP.NET Core MVC
* C#
* SQL Server
* HTML
* CSS
* JavaScript
* Bootstrap
* Entity Framework Core

## Database

## How to Run

1. Clone the repository

```bash
git clone https://github.com/Jadavvijay480/Hostel_Fronted_2026.git
```

2. Open project in Visual Studio

3. Restore NuGet Packages

4. Update database connection string in:

```json
appsettings.json
```

Example:

```json
"ConnectionStrings": {
  "DefaultConnection": "Server=YOUR_SERVER;Database=AVADH_PRIME;Trusted_Connection=True;Encrypt=False;"
}
```

5. Run the project
